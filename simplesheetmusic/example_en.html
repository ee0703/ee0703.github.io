>>>Usage

1. impoert SimpleSheetMusic.js

2. play sheet music 

var notes = [['C4',1],['D4',1],['E4',1],['F4',1],['G4',1],['A4',1],['B4',1],['C5',1]];
//create player
var player = new simple_player();
//play sheet music
player.play(notes);

3. properties

You can set 'tempo' and 'volume' properties, i'm now adding more properties like tone-colours.

var notes = [['E4',1],['E4',1],['E4',2],['E4',1],['E4',1],['E4',2],['E4',1],['G4',1],['C4',1.5],['D4',0.5],['E4',1]];
var player = new simple_player();

//Tempo(beats per min), 120 by default
player.tempo = 360;

//vilume(0.0 ~ 1.0), 0.7 by default
player.volume = 0.2;

player.play(notes);

>>> Format of the sheet music

Use Array to present notes，in format like: [['Name',beats], ...],use '0' for pause,sharp and flat should be :C#4 or Cb4. Below is an example: 

[['C4',2],['A4',1],['0',1],...]

Here's a sample for main theme of《Castle in the sky》
